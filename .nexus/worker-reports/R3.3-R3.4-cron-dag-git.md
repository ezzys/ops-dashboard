## Bug Fixed

### `src/services/git-attribution.js` — Null-byte separator crash (Node 25)

**Problem:** `getCommits()` used `\x00COMMIT\x00` as a separator embedded in the `--pretty=format:` argument passed to `spawnSync`. Node.js v25.9.0 rejects null bytes in spawn args with `ERR_INVALID_ARG_VALUE`.

**Fix:** Changed separator to a printable string `<<<COMMITSEP>>>` and moved it to a separate `%n` line in the format string. This is equally unique for splitting commit boundaries while being spawn-safe.

**Before:**
```js
const SEP = '\x00COMMIT\x00';
`--pretty=format:%H%x09%an%x09%ae%x09%aI%x09%s%x09%b${SEP}`
```

**After:**
```js
const SEP = '<<<COMMITSEP>>>';
`--pretty=format:%H%x09%an%x09%ae%x09%aI%x09%s%x09%b%n${SEP}`
```
